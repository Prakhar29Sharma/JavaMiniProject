# Appointment Booking System

## What is Appointment Booking System?
* An Appointment Booking System
* Manages Patient and Doctor Records

###

| Features                 | 
|--------------------------|
| User Registration        | 
| Check Appointments       | 
| Book Appointments        |
| View Patient Records     |
| View Doctor Data         |
| *Reschedule Appointments |

## Technologies Used :
* Intellij IDE
* JavaFX Scene-builder
* MySQL
* MySQL Workbench
